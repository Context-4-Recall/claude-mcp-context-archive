# 03 MacBook Pro + Cloud CAD/Engineering Workflow

## Supported Tools

* FreeCAD/OpenCascade for STEP and B-rep inspection;
* Onshape for browser-native CAD collaboration;
* Blender and MeshLab for mesh processing;
* CloudCompare and Open3D for point clouds;
* STEP/Parasolid-neutral exchange with SolidWorks;
* remote Windows or a governed VM when native SolidWorks-only functionality is required;
* local Python and Node execution;
* Docker Compose;
* a development container;
* local Atlas serving.
