# Model Context Protocol (MCP) Overview

MCP is Anthropic's open protocol for securely connecting Claude to external services, data sources, and tools. It allows deep, permission-aware integrations without manual tool schema writing.

Featured integrations from screenshots:
- Microsoft 365
- Atlassian Rovo (Jira & Confluence)
- Canva
- Figma
- Coda
- Subtext (session replay/debugging)
- Adobe Creative tools

See individual files for details.
